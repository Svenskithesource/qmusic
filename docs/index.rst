Welcome to Python QMusic API Wrapper's documentation!
=====================================================


QMusic API Wrapper let's you use easily use the QMusic API.
It works on Python 3.6+ and is independant of the OS.

To install with pip, run ``pip install Qmusic``. See the :doc:`install` page for more details.

The source code is available on: https://github.com/svenskithesource/qmusic
